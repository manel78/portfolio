RPG Le RPG sur un terminal de commande qui est mon tout premier projet est un jeu où le joueur interagit avec un monde virtuel en utilisant des commandes textuelles. Il incarne un personnage qui évolue dans un univers rempli de quêtes, de combats et d'exploration. Le joueur tape des commandes telles que "aller nord", "attaquer ennemi", ou "prendre objet" pour progresser dans le jeu. Les combats sont généralement gérés par des mécanismes de tour par tour, et le joueur peut améliorer les compétences de son personnage au fil du jeu.

Hangman Dans un jeu du pendu sur un terminal de commande, le joueur doit deviner un mot secret en proposant des lettres une par une. Le jeu affiche initialement des tirets représentant chaque lettre du mot à deviner. À chaque fois que le joueur propose une lettre qui est présente dans le mot secret, le jeu révèle toutes les occurrences de cette lettre dans le mot. Si la lettre proposée n'est pas dans le mot secret, le joueur perd un essai. Le jeu continue jusqu'à ce que le joueur devine correctement le mot ou épuise tous ses essais.

Groupie Trackers Réalisé en Golang et introduit l'utilisation de Fyne. Cette application offre une plateforme pratique pour les utilisateurs qui veulent suivre leurs artistes préférés, explorer leurs concerts à venir et bien plus encore. En utilisant Fyne, les développeurs ont probablement trouvé un moyen efficace de créer une interface utilisateur conviviale pour cette application de musique. Cela montre comment différentes technologies peuvent être combinées pour répondre aux besoins des utilisateurs et ouvre des perspectives sur la manière dont les applications peuvent être développées dans le futur.
